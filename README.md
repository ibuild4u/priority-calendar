# Priority Lock Calendar

A priority-based weekly calendar planner with time blocking and visual available slots.

## Features

- Priority-based locking
- Event-level durations
- Multi-day events
- Available time slots as visual guides
- Inline event editing
- Theme support (Default and Soft Pink)
- Local storage persistence
- Export to PNG, SVG, and JSON

## Installation

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`

## Deployment

Run `npm run build` then deploy the `dist` folder to GitHub Pages.
